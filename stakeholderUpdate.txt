  - `optumIOA`

3. What Is Dynamic Already

- Sidebar selection is dynamic and route-driven.
- Parent and child navigation states change automatically based on the page URL.
- The top filter strip is reusable across pages.
- Overview KPI cards are data-driven from arrays, not hardcoded block by block.
- KPI value colors are dynamic:
  - positive HCC capture values show in green
  - negative trend values can show in red
  - value-per-visit uses green styling
  - estimated EOY savings uses dark yellow styling
- YTD tables are data-driven from row arrays/objects.
- Highlight colors in YTD tables are dynamic by row type:
  - green
  - yellow
- The same page pattern is reused for both Signify HHV and Optum IOA.

4. UI Refinement Work Completed

- Sidebar background and spacing improved.
- Sidebar text hierarchy improved for:
  - main module names
  - sub-module items
- Sidebar icon placement and alignment improved.
- Main content spacing improved.
- Overview filter strip was changed from a card-style block to a plain page-level strip.
- Summary cards were updated to better match the provided screenshots.
- YTD tables were visually aligned to the reference design:
  - table title row
  - blue column header row
  - gray metric column
  - green/yellow value highlights
  - balanced table height and spacing
- YTD table styling was moved into dedicated page CSS files for cleaner separation.
